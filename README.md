UnlockWord
==========

Android lock screen application in order to memorize words.

已解决问题：
 1. 与4.0系统的兼容。
 2. 干扰答案随机。
 3. 开机启动。
 4. 显示音标。

待解决问题：
 1. 锁屏Activity启动时，关闭屏幕，Service又一次启动Activity。
 2. 关闭屏幕时，来电应答的显示。

考虑：
 1. 通过Service注册屏幕变暗的广播能否与用户进入的广播结合使用。

以后要加的功能：
 1. 先答错后答对了突出显示一下，强化记忆。
 2. 加上帮助页。
 3. 可以选择词库。
 4. 从网络下载新词库。
 5. 例句。
 6. 多种题型。
 7. 读音。
